# MicroTools

**Free AI-Powered Micro Utilities** — small tools that do one thing well.

All tools run 100% in your browser. No backend, no data collection, no sign-up required.

🌐 **Live:** [beelinkfarm.github.io/micro-tools](https://beelinkfarm.github.io/micro-tools/)

---

## Tools

### 🔍 LLM Scraper
Convert any webpage into clean, LLM-ready markdown text. Strip noise, extract signal.

→ [Try it](https://beelinkfarm.github.io/micro-tools/llm-scraper.html)

### 🔖 Bookmark Organizer
Auto-categorize and analyze your browser bookmarks with AI. Bring order to the chaos.

→ [Try it](https://beelinkfarm.github.io/micro-tools/bookmark-organizer.html)

### ⚡ MultiSearch
Search across 5 engines (Google, Bing, DuckDuckGo, Brave, Perplexity) simultaneously. Compare results, find more.

→ [Try it](https://beelinkfarm.github.io/micro-tools/search-compare.html)

### 📊 DataView
Paste JSON or CSV data, instantly visualize in sortable tables, edit inline, filter, and export.

→ [Try it](https://beelinkfarm.github.io/micro-tools/data-viewer.html)

### 🔗 CardPreview
Generate and preview social media cards for any URL. See how your link looks on Twitter, Facebook, LinkedIn, Discord, and Slack. Export OG meta tags.

→ [Try it](https://beelinkfarm.github.io/micro-tools/url-preview.html)

### 📡 HN Watch
Monitor Hacker News threads for new comments. Watch discussions, get browser notifications, never miss a reply.

→ [Try it](https://beelinkfarm.github.io/micro-tools/hn-monitor.html)

---

## Features

- **100% Client-side** — Everything runs in your browser. Your data never leaves your device.
- **No sign-up** — Just open and use. No accounts, no tracking.
- **Dark/Light theme** — All tools support both themes.
- **Mobile responsive** — Works on any device.
- **Free & Open** — All tools are free to use.
